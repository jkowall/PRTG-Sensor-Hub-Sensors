# PRTG-Sensor-Hub-Sensors

This is the repository of sensors for the PRTG Sensor Hub.

## Overview

This repository contains custom sensors that can be used with the PRTG Sensor Hub (see [@jkowall/PRTG-Sensor-Hub](https://github.com/jkowall/PRTG-Sensor-Hub)). Sensors are organized by category in the `sensors/` directory.

## Structure

Sensors are organized into the following categories:
- **Network** - Network-related sensors
- **Other** - Additional sensors for various use cases

Additional categories will be added as sensors are added to sensor hub.
